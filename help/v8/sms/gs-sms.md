# 开始使用短信投放 {#sms}

短信投放提供了一种实用而高效的方式，可将短信发送到客户的移动设备。 利用此功能，您可以创建、个性化和预览基于文本的消息以进行有效通信。

短信渠道提供了三种创建短信投放的方法：

* 工作流：添加短信活动，定义基本设置，并在更大的自动化序列中撰写短信内容。

* 营销活动：将短信投放合并到专用营销活动中，并设置目标受众和投放计划等参数。

* 独立投放：创建单个短信投放，以便与客户快速而直接地进行通信，而无需工作流或营销活动。
