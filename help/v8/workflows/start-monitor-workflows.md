# 启动和监视工作流执行 {#start-monitor}

内容待定

* 如何运行工作流
* 实时跟踪其执行情况。

   默认情况下：活动状态显示+过渡结果。 隐藏进度按钮以隐藏

   蓝色可视流

* 暂停/停止工作流。 暂停后继续工作流。
* 访问日志和任务
* 出错时会发生什么情况：可视指示器+状态

   <!--to reformulate-->您可以配置工作流，以便在发生错误时不会暂停并继续执行。 要执行此操作，请编辑工作流属性，并在错误管理部分中，选择“发生错误”字段中的“忽略”选项。 然后，您可以指定在暂停进程之前可忽略的连续错误数。
